## Project Description

https://isabelabaseggio.github.io/Hex-Color-Test/

Hex Color Test is a mini web quiz aimed at practicing programming logic, functions and responsive layout.

The quiz consists of drawing a hex color code each round as a question, with four color options as a multiple-choice answer, of which only one corresponds to the hexadecimal code of the question. At the end of the quiz, after ten rounds, the player's score will be shown, as well as the option to restart. Furthermore, each time the question is answered it will be indicated in hexadecimal code with a green border if the answer is correct, if it is wrong the border will be red.
